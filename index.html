elige el nivel del listening que has de escuchar

A
Principiante / Esencial

Para estudiantes que están construyendo su base. Los videos tienen un ritmo claro, vocabulario cotidiano y estructuras sencillas para acostumbrar el oído sin frustrarse.

B
Intermedio / Independiente

Para quienes ya entienden la idea general pero quieren defenderse con nativos. Ideal para captar modismos comunes, velocidad de conversación normal y acentos variados.

C
Avanzado / Experto

Para estudiantes que buscan el dominio total. Diseñado para entender *slang* pesado, chistes rápidos, referencias culturales complejas, voces distorsionadas y clips de series sin filtros.
